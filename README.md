# python3_webapp_littleproject
my first python3 project, learn from liaoxuefeng's website

orm：ORM
主要负责的是数据库方面的操作，可以方便的让python对象实现数据库的增删改查操作
该模块封装了aiomysql库


coroweb：Web框架
主要负责的是接受http请求和处理请求并返回结果，
该模块可以保证开发者用最少的最简明的代码实现上述功能功能
该模块封装了aiohttp库
